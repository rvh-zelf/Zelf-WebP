== Description ==

This plugin uses the Spatie Image Optimizer package for compressing and converting images to WebP format. WEBPs will be optimized by Cwebp.

Simply activate the plugin and all newly uploaded images will be automatically optimized and converted to WebP format.

== Frequently Asked Questions ==

= How does it work? =

The plugin will automatically optimize and convert images to WebP format when they are uploaded to the media library.

= What about existing images? =

Existing images will not be touched. Only newly uploaded images will be optimized and converted.

= What if I already have WebP images? =

If you already have WebP images in your media library, they will not be touched. Only newly uploaded images will be optimized and converted.

= What if I want to convert existing images? =

This is not currently supported.
